# Scooter API Test

Этот проект я сделала для автоматизации проверки создания и получения заказов через API Яндекс.Самокат.

## Автотест "Создание и проверка заказа"

**Описание:**  
Написала тест, который проверяет, что клиент может создать заказ через API и затем получить его по треку.

**Шаги теста, которые я выполнила:**
1. Отправила POST-запрос на создание нового заказа.
2. Сохранила трек-номер созданного заказа.
3. Отправила GET-запрос для получения заказа по треку.
4. Проверила, что код ответа равен 200.

**Результат:**  
Если код ответа 200 → тест пройден, заказ создан и доступен для получения.

**Пример вывода в терминале:**
Заказ создан. Трек: 920293
Тест пройден! Код ответа 200
